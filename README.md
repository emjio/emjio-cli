# emjio-cli
a cli for vuepress whit theme

## Version


## Experience

```bash
# create
npx emjio-cli init my-blog
# oa
npm install  -g emjio-cli
emjio-cliinit my-blog

# install
cd my-blog
npm install

# run
npm run dev

# build
npm run build
```

**if yarn**

```bash
# create
npx emjio-cli init my-blog
#or
yarn global add emjio-cli
emjio-cli init my-blog

# install
cd my-blog
yarn install

# run
yarn dev

# build
yarn build
```
